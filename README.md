# Ragebait-chess aka Aichess 
This is a website where you can train chess with ai powered opponents, look at your statistics and most importantly have fun...
_for free!_
This project is **being** built by a group of 4 using SCRUM:
- Remi, project owner and frontend focused developer
- Eeli, SCRUM master and fronted focused developer
- Leo, backend focused developer
- Oleg, backend focused developer

We will be using react with Vite, and Express.js to build pur project, and tools like Figma, Trello and draw.io to visualize the project.
